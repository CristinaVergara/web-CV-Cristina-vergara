Hola!

Soy diseñadora gráfica con orientación UX/UI, apasionada por crear experiencias visuales claras, funcionales y emocionalmente conectadas. A lo largo de los años he explorado diversos lenguajes visuales —desde el diseño editorial y la identidad de marca, hasta la arquitectura de interfaces y la interacción digital— siempre buscando balance entre estética y propósito. Actualmente me encuentro formándome en desarrollo web front-end, explorando con entusiasmo tecnologías como HTML, CSS, Flexbox, Bootstrap y WordPress. Este nuevo lenguaje técnico me permite complementar mi enfoque visual con herramientas para construir productos digitales integrales, donde el diseño y el código dialogan con coherencia. Me inspira observar el comportamiento de los usuarios, estructurar soluciones que se adapten a sus necesidades y transformar ideas en interfaces limpias, responsivas y sensibles al detalle. Mi proceso es una constante búsqueda entre lo emocional y lo funcional —porque para mí, el diseño también se siente.

Portfolio

Primer Desafio: CV en html

Como primer paso en mi formación en desarrollo web front-end, construí un currículum vitae utilizando únicamente HTML. Este ejercicio me permitió aplicar conceptos fundamentales como la estructura semántica, el uso correcto de etiquetas (header, section, ul, article, etc.), y la organización jerárquica del contenido. El objetivo fue crear una página clara, accesible y bien estructurada, que reflejara tanto mi perfil profesional como mis primeros conocimientos en maquetación web. Este proyecto marcó el inicio de mi transición desde el diseño visual hacia el desarrollo de interfaces funcionales.

Landing page deco bebe

Como parte de mi proceso de aprendizaje en desarrollo web front-end, diseñé y construí una landing page para una marca ficticia de decoración infantil. El objetivo fue crear una experiencia visual cálida, clara y responsiva, aplicando conceptos de diseño UX/UI y maquetación con HTML y CSS.

Iguana Page

Este proyecto consistió en replicar una maqueta visual utilizando exclusivamente HTML y CSS, aplicando los conceptos fundamentales de maquetación, tipografía, estructura semántica y estilos personalizados. El objetivo fue construir una landing page estática, cuidando la jerarquía visual, el uso de fuentes desde Google Fonts, íconos con Font Awesome y una paleta cromática definida. Durante el desarrollo trabajé con una estructura de carpetas ordenada, integrando imágenes, estilos y contenido de forma modular. Este desafío me permitió afianzar el uso de Flexbox, clases reutilizables y buenas prácticas en la organización del código, consolidando mi base técnica en el proceso de aprendizaje front-end.

Landing Page cuppon

En este desafío desarrollé una landing page responsiva para una plataforma de cupones, aplicando el sistema de grillas de Bootstrap y sus componentes principales. El objetivo fue construir una interfaz clara, ordenada y adaptable a distintos dispositivos, utilizando tarjetas (cards), formularios y clases utilitarias para estructurar el contenido de manera eficiente. Este proyecto me permitió profundizar en el uso de Bootstrap como framework de maquetación, mejorar la organización del código HTML y CSS, y reforzar buenas prácticas de diseño responsive. También trabajé en la jerarquía visual, la consistencia tipográfica y la experiencia del usuario en dispositivos móviles.

Fork y personalización de sitio web

En este desafío trabajé con GitHub para forkear un repositorio existente, clonar el proyecto localmente y modificar el código de una web básica, transformándola en una versión personalizada que reflejara mi identidad profesional. Apliqué conocimientos de HTML y CSS para adaptar la estructura, los estilos y el contenido, incorporando elementos visuales propios y una narrativa coherente con mi perfil. Durante el proceso realicé múltiples commits para registrar los cambios de forma ordenada, fortaleciendo mi comprensión del control de versiones y las buenas prácticas de trabajo colaborativo. Este ejercicio me permitió integrar diseño y desarrollo en un flujo real de trabajo, utilizando herramientas clave del ecosistema front-end.

Post it Colaborativos

En este desafío trabajamos en equipo para construir una web colaborativa tipo “muro de post-its”, utilizando Git y GitHub como herramientas principales de control de versiones. A través de la plataforma, aprendimos a forkear un repositorio, crear ramas (git checkout -b), realizar commits, y sincronizar cambios con git push y git pull. El proyecto nos permitió experimentar con flujos de trabajo reales, resolver conflictos de fusión y comprender la importancia de una buena comunicación entre colaboradores. Cada integrante aportó su propio bloque de contenido, integrando diseño, código y organización en un entorno compartido. Esta experiencia fortaleció mis habilidades técnicas y mi capacidad para trabajar en equipo dentro de un ecosistema de desarrollo profesional.
